# mysql-to-mongodb-mirgation
2020-1 Database Application : RDBMS(MySQL) 데이터 Document Data Model(MongoDB) Mirgation Project

## Build
```
$ pip install requirements.txt
```

## Execute
```
$ python3 mysql_to_mongo.py 
```
